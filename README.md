# Icedog-Loadscreen (v1.3)

# Simple custom loading screen I’ve been working on for a old server. Still a work in progress but functional and customizable.

# ✨ Features

Animated loading progress bar
Rotating tips system
Custom backgrounds & logos
Smooth modern UI animations
Easy config customization
Lightweight & optimized

# ⚡ Setup

Drag the folder into your server resources
# Comment out/remove your old loadscreen from server.cfg or resources.cfg
```# ensure old-loadscreen```
# Add this to your server.cfg or resources.cfg
```ensure icedog-loadscreen```

Then just restart your server!
